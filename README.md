# QuanLiQuanTS

Ứng dụng Android quản lý thông tin khách hàng, nhiệm vụ, nguyên liệu và nhân viên.  
Dự án được xây dựng để thực hành lập trình Android.

## Chức năng

- Đăng ký, đăng nhập
- Bật / tắt âm thanh
- Xem danh sách khách hàng, nhiệm vụ, nguyên liệu, nhân viên
- Thêm, sửa, xóa, tìm kiếm thông tin
- Thêm người dùng
